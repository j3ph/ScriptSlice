Easily include externally hosted scripts inside the header or footer section of your wordpress website. No need to edit any files!

== Description ==

This plugin allows you to easily include an externally hosted script inside any and every page withing your WordPress website. The interface let's you paste the script once then include it on the page your need. You can even choose to include on every page. 

Need to place the script inside your head section? No problem. Need one to load at the bottom of the page? It can do that too :)

== Installation ==

1. Upload `script-slice` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings -> Script Slice to get started!
